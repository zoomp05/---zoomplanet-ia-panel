## üé® Layouts y Herencia

- `/zoomy/admin/auth/login` ‚Üí Usa `MainLayout.jsx` de Admin

## ‚úÖ Checklist de Migraci√≥n

Para migrar un site existente al nuevo sistema:

- [ ] **1. Crear `site.config.js`** con definici√≥n de todas las instancias de m√≥dulos
- [ ] **2. Agregar `routing` config** a cada instancia de m√≥dulo
- [ ] **3. Actualizar `sites/{site}/index.js`** para usar `ModuleInitializer`
- [ ] **4. Mantener** `registerSiteRoutes` y `registerModuleRoutes` existentes
- [ ] **5. No modificar** los `routes/index.js` de cada m√≥dulo
- [ ] **6. Configurar** `parentModule` para definir jerarqu√≠a de rutas
- [ ] **7. Definir** `routePrefix` si se necesita prefijo personalizado
- [ ] **8. Verificar** que las rutas se generen correctamente con `debugRouteTree()`
